# Bright Side Study

A modern, responsive website for Bright Side Study - Global Education Counsellor.

## Features

- Responsive design for all devices
- Modern React + Vite setup
- Beautiful UI with custom CSS
- Comprehensive sections for education counselling services
- Contact form for student inquiries
- Information about UK and international study destinations

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
